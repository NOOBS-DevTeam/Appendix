Структура программы

В любой программе на языке apx должна быть определена функция int main(), которая является входной точкой в программу. Возвращаемым значением должен быть код возврата программы - целочисленное 4 битное число.

Инструкции отделяются символом точка с запятой. Блоки кода должны быть ограничены програмными скобками, роль которых выполняют фигурные скобки. После последней инструкции в блоке не должно быть точки с запятой. Если блок кода находится внутри функции, то после него должен стоять  символ точка с запятой.
